# Projects Summative Lab!
   This summative lab has been done to showcase a few projects as well as have the user 
   add their own projects to the projects listing.
- the most important documents the src folder contains are the mainApp.jsx and style!.css.
- the code for displaying projects is within the components folder - in src.
## Issues:
  There are a few issues that are yet to be resolved within this project. 
   - first, when adding a project, the img addition does not display.
   - second, the border formatting displays oddly when done to the added project,
      causing an uneven format so I have chosen not to include it.
   - third, adding more than one project. Instead of adding the next, it updates the first.
### To work on the project: 
Git clone this repository into your system using Ubuntu
- Run the code on yout chosen platform, (I used VS code) and write in the terminal
" npm run dev ".
- This command will give you a local host for you to copy or click the link to 
go check the website.
